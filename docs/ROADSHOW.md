> **这是整个路演的核心环节，务必提前准备好现场演示环境。**

### 演示 1：浏览器自动化（1.5 分钟）

在飞书中发送：

```
帮我打开京东，搜索"蓝牙耳机"，采集前 10 个商品的名称和价格
```

**观众看到的效果**：
- 浏览器自动打开，鼠标像真人一样移动（贝塞尔曲线轨迹）
- 搜索框逐字输入（随机打字延迟，不像机器人）
- 自动滚动采集数据，输出结构化表格

**讲解要点**：
> "注意看鼠标轨迹和输入速度，这不是简单的 HTTP 爬虫，而是 RPA 级别的浏览器自动化。13 项反指纹措施确保不会被反爬拦截。"

### 演示 2：记忆 + 自我学习（1.5 分钟）

```
帮我记住：张经理的邮箱是 zhang@company.com，他喜欢周五下午开会
```

等回复后，再问：

```
张经理的邮箱是什么？
```

**讲解要点**：
> "灵雀不是简单地保存文本，而是自动抽取实体和关系存入知识图谱。下次问到相关内容时，会从记忆中自动召回。而且每次任务结束后，灵雀会自动反思——哪些方法有效，哪些错误下次要避免。"

## 第 4 页 · 核心技术亮点（2 分钟）

### 五大技术创新

**1. 自我进化引擎** 🧠
```
任务完成 → 自动反思经验
工具失败 → 记录错误 + 修复方式
用户反馈 → 捕获"很好"/"不对"信号
下次任务 → 自动召回相关经验
```
> "灵雀越用越聪明，这不是营销话术，是实打实的技术实现。"

**2. 三层记忆 + 知识图谱** 📚
- 短期：会话上下文（80 条消息 / 64K tokens）
- 长期：MEMORY.md + 知识图谱（实体-关系三元组）
- 检索：SQLite FTS5 / ChromaDB 向量语义搜索

**3. 用户画像 + 情绪感知** 👤
- 自动识别用户兴趣、技术水平、使用习惯
- 实时情绪检测，根据用户状态调整回复风格
- 成长里程碑系统，陪伴用户一起成长

**4. RPA 级浏览器自动化** 🌐
- 13 项反指纹（Canvas/WebRTC/AudioContext/WebGL...）
- 贝塞尔曲线鼠标 + 人类化输入 + 滑块验证码自动破解
- Chrome 扩展增强：深度 DOM 分析 + 多策略元素定位

**5. 9 层纵深安全防护** 🔒
- 从通道鉴权到沙箱隔离到审计日志，企业级安全
- 凭证保险箱：API Key 安全存储，不进 .env，不进记忆，日志自动脱敏

## 常见评委提问 & 参考回答

**Q: 和 Dify/Coze 的区别是什么？**
> 它们是 SaaS 平台，数据上传云端；灵雀是私有化部署，数据不出本地。而且灵雀有自我进化引擎和 RPA 级浏览器自动化，这是它们没有的。

**Q: 数据安全怎么保证？**
> 9 层纵深安全防护：从通道鉴权、操作分级、文件隔离、命令审查、密钥保险箱到 Docker 沙箱，全链路安全。所有数据存在用户自己的服务器上，不上传任何第三方。

**Q: 自我进化是怎么实现的？**
> 三个机制：任务反思（每次任务结束后 LLM 总结经验）、错误修复配对（记住错误和修复方式）、用户反馈捕获（识别正面/负面评价）。经验存入检索引擎，下次类似任务自动召回。

**Q: 目前有多少用户？**
> 目前在内测阶段，已有 XX 个团队在使用。开源版本刚上线，正在积累社区用户。

**Q: 商业化路径？**
> 短期：开源获客 + 企业私有化部署收费。中期：技能市场生态 + 行业解决方案。长期：成为私有化 AI Agent 的标准框架。
